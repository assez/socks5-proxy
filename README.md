## 📢 选代理IP前必看（省掉麻烦，使用更顺畅）

### ① Socks5 / HTTP / L2TP 如何选择？
- **80%场景用Socks5**：线路成本低，配置方便，无需硬件
- **项目或游戏防控强**：可选 L2TP，需搭配软路由，成本较高，适合多设备便捷使用
- **苹果手机用户**：如无法下载小火箭，可选 L2TP，无需下载即可配置

### ② 带宽与地区如何选择？
- **带宽非下载速度**：10M带宽理论下载速度约 **1MB/s**，实际在0.8~1MB/s之间
- 根据项目选择：挂机游戏10M基本够用；网页浏览、自媒体、矩阵、电商建议越大越好
- **地区选择**：无特殊要求时，选自身周边地区，相对稳定且损耗低

### ③ 住宅IP与机房IP的区别
| 类型 | 来源 | 特点 | 适用场景 |
|------|------|------|----------|
| **住宅IP** | 家庭宽带运营商（电信/联通） | 真实度高，不易被封禁，需定期维护 | 社交媒体管理、广告验证、游戏多开等需高度模拟真实用户 |
| **机房IP** | 数据中心或云服务商（阿里云/AWS） | 稳定性高、带宽大，易被识别为数据中心IP | 数据采集、服务器管理、API调用等高并发、大流量业务 |

#### 🧠 总结
住宅IP重真实、易维护；机房IP重稳定、易屏蔽。根据项目对"真实性"和"稳定性"的侧重选择即可。
